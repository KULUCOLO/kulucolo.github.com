猫的报恩/猫の恩返し/The Cat Returns

发表于 2015-04-26 | 分类于 小草熊编 ， 2000~2010 | | 阅读次数

小草熊的说：

第一次在《侧耳倾听》中看到了条推荐《猫的报恩》的弹幕，于是就一直记着啥时有机会去看一眼，后来去B站再找的时候发现宫老的作品都挂了，此刻我内心是悲伤的，于是就搁在了一边。昨天第一次打开这个02年的作品，感觉还是很新鲜。首先是画风，猫的报恩画风比起之前所看过的吉卜力有点新鲜的感觉，我从不同角度看女主仿佛她的脸型时刻都是变化的，有股说不上的异常却又不是很变扭，感觉有点萌。然后接着说喵星人，小编我从未见过如此拟人的喵星人，从喵国王的保镖，到喵国王的参谋和军师，还有喵王子和喵女仆，最后喵男爵，哇，一整个下来完全就是一出精致的童话。

介于没有过多的给单身狗造成伤害，因此强烈推荐给所有小孩子们和还存有颗童心的大孩子们。

《猫的报恩》（猫の恩返し，中国大陆、台湾、新加坡译作《猫的报恩》，港澳地区译作《猫之报恩》）是吉卜力工作室2002年推出的一部动画电影。故事改编自漫画家柊葵的漫画。于2002年第6回文部省文化厅媒体艺术祭获选动画部门优秀奖。

与吉卜力另一部短篇动画《Ghiblies episode2》，为同时在戏院上映的作品。@Wikipedia

故事简介

内容描述一位平凡善良的高中女生小春，在一次放学途中救了一只猫，意外开启了一趟奇幻冒险经历。小春救的是猫王国的王子，“猫王国度”的猫国王为了报答她，不断送她礼物，并邀请小春到猫王国作客，希望她嫁给猫王子。无助的小春向“猫咪事务所”的猫男爵（バロン）求助，遇到肥猫胖胖（ムタ）、乌鸦多多（トト）。突然，小春被一大群猫强行带往猫国，猫男爵和胖胖、多多紧追进入猫国。待在舒服的猫国中，小春渐渐地长出胡须和猫耳朵，小春快变成真正的猫了。在猫王的阻挠之下，小春、猫男爵和胖胖拼命的奔往迷宫出口，想让小春在天亮之前重回人类世界，否则将永远做猫。@Wikipedia
