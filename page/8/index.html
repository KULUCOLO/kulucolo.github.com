夏日大作战/Summer Wars/サマーウォーズ [1080P]

发表于 2014-03-31   |   分类于 柚编 ， 2000~2010   |     |   阅读次数

柚编的说：

不用惊讶，这次又来了一个叫柚编的家伙

拯救世界不需要有超能力，也不需要内裤外穿，只要有一个足够庞大的大家庭就够了！（真的吗？）

乍一听，这部影片似乎是满满的动感（实际上也的确如此），但欢脱的节奏中细田守至始至终保持着他那股独特的清新，在虚拟化的OZ世界中，也同样能让人产生真实的感受，没错，继穿越时空烧酒之后，细田守又再一次成功地引起了观众们对人生的共鸣。亲情，友情，爱情，人与人之间的信任，这便是夏日的围绕着的主题。

影片之中吸引小柚编的还有浓郁的家的味道。在这个大家庭中，有孕育中新生婴儿，有做错事熊孩子，有和蔼的长辈，也有逝去的人。生命不断轮回，而自人们离去的那一天起唯一能延续的就是一种精神和一个态度。太婆婆无疑成为该影片中的那颗最闪耀的星，乐观的她维系着整一大家庭，因为几乎从每个人的身上都能够看到她那个微笑的影子。

玲音/Serial Experiments Lain

发表于 2014-03-29   |   分类于 小草熊编 ， 1990~2000   |     |   阅读次数

小编的说：

玲音是小编的朋友最近推荐的一部作品，抛开关注度不说，单从她的试验片上面就足以列入KuluColo的推荐中来。应该是受到了EVA的意识流影响，人们发现动漫不再单纯是讲述一个童话故事，更能作为一种意识载体，投向社会中来。于是，在这部试验性动漫中，晦涩而含蓄的赛博朋克思想始终贯穿着Lian的整个篇幅，不是单纯的记录着故事，Lian更多的讨论了人类的意识与人之间的关系，当然从其中也可以一窥98年的人对未来网络，未来计算机的前瞻性的认识。

小编看到了Lian的一个标签是“庵野秀明八成也看不懂”XDDD，能让我们深刻意识到什么叫做意识流，也作是Lian的一大亮点了。

我们仍未知道那天所看见的花的名字/あの日見た花の名前を僕達はまだ知らない [1080P]

发表于 2014-03-27   |   分类于 豆豆编 ， 2010~2020   |     |   阅读次数

豆豆编的说：

“无论与你共度的夏天，将来的梦，还是庞大的希望 我决不忘怀，相信10年后的8月 我们还能够遇上彼此，怀着最美的回忆…”

其实豆豆编知道《我们仍未知道那天所看见的花的名字》纯属意外，只是刷空间的时候看到一个小伙伴说“又一次看哭了。”于是心怀好奇地搜了这部片。一开始看的时候并没有很大的感触只是觉得OP里的过去与现状的穿插实在是赞，到后来慢慢被吸引了，想说那种清新的感觉真的很好。

小时候，一个回不去的地方，其实，那个时候真的很好。天永远是蓝的，没有什么可以称为烦恼的东西，这边晃晃，那边走走，磨磨蹭蹭地背起书包和三两好友说说笑笑。人总会有时就突然陷入对往事的回忆里，那些或许清晰，或许模糊的东西，在岁月的冲洗与美化下成为无比美好的东西。“为了守护世上所有的和平”为目的而组成了“超和平Busters”，从孩子时期就一直在一起的六个主角因为面麻的出事解散，大家为了实现面麻的愿望重聚在一起可是各自却又怀着对过去深深的放不下的小小心思，到底结局如何就留待各位看官自己揭晓了. . .同时豆豆编不得不感慨句，面麻是个好姑娘阿~我都有直接冲进去给她下碗鸡蛋面的冲动。她真的是很可爱的女孩子，也很善良。或许，也正是因为她的性子，那种天然的美好的东西，她的伙伴们，才会一直都无法走出羁绊吧。最后，仁太对面麻喊道“你在玩捉迷藏吧，那么不把你找到就无法结束了吧！”我们都知道这句孩子气的话里藏着的不舍，留恋等等的情愫，六个小伙伴一起喊着“好了吗”而在尾声，捉迷藏结束了“我藏好了。”仁太，安鸣，鹤子，雪集，波波。那时候呐，阳光照了下来，不猛烈，但很舒服，真的。

“所谓道别，可不是单方面的说说”，什么叫道别，什么叫释怀，什么叫友情，什么叫回忆，就先知道那朵花的名字吧。
