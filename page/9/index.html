穿越时空的少女/時をかける少女 [1080P]

发表于 2014-03-18 | 分类于 柚编 ， 2000~2010 | | 阅读次数

柚编的说

能穿越的不一定是哆啦A梦的时光机，也有可能是「Time Leap」超能力；带给你微笑的不一定是红豆铜锣烧，也可以是“我在未来等你”的年少的美丽约定。骑白马的不一定是王子，不要忘了唐僧。

蝉鸣，树影，白云，夏天，钢琴声响，棒球手套，骑车上学时的凉风，傍晚放学后的夕阳，图书馆书架间微微的凉意，篮球场里永无休止的喧嚣。十七岁的我们，正直青春，时光奔涌不止，我们却有时光不变、小伙伴们永远不会分开的错觉。

小编看《穿越时空的少女》的时候正是高中，如今再回首，觉得每个人的生命中都有一些截点。回忆到了那里，就停住了。而我们总是在往前走，一部分的你被时光拦截下来，她和你挥手告别，永远的留在那里。剩下的部分，总是试图回头去寻找那些碎片。

《穿越时空的少女》保留的就是那些珍贵的碎片：那种不随时代改变的青春感；以及隐没在轻松跳跃剧情下的，对青春深刻又真切的思考。我们以为的青春可以尽情挥霍可以不断重来，事实上转瞬它就落在了回忆里。

细田守是本片的导演，《夏日大作战》、《狼的孩子雨与雪》都出自他手。或许更能勾起各位回忆的应该是两部《数码宝贝》剧场版：《光丘事件》（《滚球兽的诞生》）、《我们的战争游戏》。

清新。这是细田守的关键词。不论是《超越时空的少女》那样青涩的清新还是《夏日大作战》那样欢快的清新，清新总是贯穿他的作品始终。只要见到那熟悉的细田守式的画面和贞本义行的人设，清新就扑面而来。人生，这是细田守作品的另一个关键字。细田守非常擅长描绘画卷式的人生片段。细田守最钟爱、最擅长表现的——那些带着些许小科幻的关于人生片段的清新故事。

细田守的魔力就在于，观影后的兴奋喜爱感慨感动之余，你总是能或多或少把自己的经历和影片联系起来，会觉得很有共鸣，进而自己的经历和生活都变得清澈起来，泛着粼粼波光。

如果能穿越时空，你想去哪里？是遥远的外太空，还是中世纪的欧洲，亦或是看看未来的自己？我只想回到那些时光的截点，拥抱懵懂无助的自己，告诉那个孩子：没什么的，我在未来等你。

四叠半神话大系/四畳半神話大系/The Tatami Galaxy [1080P]

发表于 2014-03-17 | 分类于 小草熊编 ， 2000~2010 | | 阅读次数

小编的说：

四叠半神话大系，汤浅政明的作品之一，小编当年光看名字就觉得有些很厉害的样子，后来了解才知道，畳（又作叠）日文读作TATAMI，就是榻榻米的意思，所以这部动漫也就围绕着男主四叠半的宿舍/围绕着男主丰富多彩的大学生活而展开。时而让人吐槽不能唯独张着嘴巴，又或者忧郁直至宅们内心的最深处。片中，主人公不算多，除了有男主“我”以外，“我”的好基友是一个有长得很奇怪的小津，以捉弄情侣为乐（简直像恶魔一般啊），师妹明石则是冷艳的黑发少女，同时也是该片的女主。另外还有，小津的“师父”，醉后喜欢乱舔别人脸的牙科护士，竞争对手城崎，人偶香织，算命老婆婆（这个绝对是重要线索）等其他的人物也共同组成了这一“神话”。整个世界观庞大而有趣，绝对说是一部可以让观众一直保持新鲜感的动漫。

小编自己是很喜欢这种类型的番，轻松惬意，既不卖肉也不商业化，它更接近生活，更能引起共鸣，仿佛就在我们身后，转个身就能抓住似的。因此强烈推荐！
