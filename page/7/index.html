我们的存在/僕等がいた [DVDRIP]

发表于 2014-04-11 | 分类于 豆豆编 ， 2000~2010 | | 阅读次数

豆豆编的说：

喜欢一个人是什么感觉？“地球不会为一个人而转动，但是感觉你就是专门为了我而存在的。”

《我们的存在》的故事就是这样从一个平凡无奇的高中生高桥七美和一个极受欢迎的高中矢野元晴之间展开。看起来的确也是这样的传统少女漫画的套路可是不得不说对人物心里的揣摩实在出色。暗恋一个人的时候，那种忐忑，只要说一句话就开心地像是世界转瞬晴天。即使分开，即使背叛，她在你心里的位置还是那样地让你无法忽略。在接近矢野的过程中，高桥发现了隐藏在矢野闪光背后的悲伤的回忆——矢野死于车祸的前女友奈奈，而奈奈的妹妹山本也卷入其中。逐渐地，矢野的好友竹内也被高桥所吸引。高桥与矢野也因为过去的事情而彼此痛苦地互相思念。其实，有时候我们总会遇到这么一个人，让你傻傻地笑，傻傻地哭，却始终无法将目光从他的身上移开。或许，他有着并不轻松的回忆，所谓的前任，像是鸿沟架在两人之间，但是请记住过去的只不过是和梦一样虚幻的东西，把握现在，创造两个人的未来才是上上签。如果你正徘徊在恋情的交叉口，或者期待着一份真实可爱的恋情，又或是刚刚离开所深爱的人，相信《我们的存在》能够告诉你我们的存在究竟是什么，即使微渺，即使卑微，我们的存在必定会遇上一个让他变得充满意义的人。

虫师/蟲師/むしし/Mushishi [720P]

发表于 2014-04-10 | 分类于 豆豆编 ， 2000~2010 | | 阅读次数

豆豆编的说：

很多人一上来就喜欢将《虫师》和《夏目友人帐》做对比，但事实上两部并不是同一个类型的漫。相比于《夏目》的随意与温情，《虫师》更多的则是探索与思考，关于感情，关于自然，关于人。《虫师》的总体节奏是很慢的，就像OP一样，很适合在疲惫的时候看，很是治愈。《虫师》的画风很简单却有种精致感，主人公银谷永远穿着那现代感十足的白衣服，背着个药箱，行走在山上，水边，乡间的小路上，帮助人们解决各种关于虫的问题，这些便是虫师的工作了。

虫师中的虫，不同于自然界的昆虫，而是比植物更原始的物种，是与生命根源最近距离的存在。当 随时准备着截屏作壁纸、非要带上耳机才肯接着往下看 成了习惯，豆豆编才发现过去看动漫的方式都弱爆了。人总是在不断行走的，途中，你会遇见各种各样的风景，各种各样的人和事，这是一种阅历，更是一种与世俗不同的超凡乐趣。 如今，虫师似乎已经不能再算是冷番了，但豆豆编我还是希望在点开《虫师》的时候，依旧怀着一颗平静地心，好让那一个个看似神奇却又无比贴近生活的故事在那26个20多分钟里向你娓娓道来。

（结尾的表里大家一定能看到风动画制作的名字，不要不相信，虫师可是有中国内地动画公司所参与制作的哦~）

她与她的猫/彼女と彼女の猫/She and Her Cat [DVDRIP]

发表于 2014-04-08 | 分类于 柚编 ， 1990~2000 | | 阅读次数

柚编的说：

《她和她的猫（彼女と彼女の猫）》——新海诚之作。

一部五分钟的动画，没有音乐，没有颜色，甚至没有对白。一个平平淡淡的声音向我们讲述了一个普普通通的美丽上班女和她的猫的故事。故事没有第四个角色，大概那只叫咪咪的猫便可以算作是第三个了吧；故事也没有起伏，只有一个电话算是故事的高潮。然后女人离开了。那会是一个怎样的电话，女人又为何对着猫哭泣。

这应该是一个每天都会发生的故事，一只叫咪咪的小猫喜欢会思考的公猫，而公猫喜欢着成熟的女主人，女人或许又爱恋着远方的男人，那男人呢？是不是喜欢着另一个爱猫的女人？我们每个人都可能是其中的角色，那只母猫，那只公猫，那个女主人，或者电话另一头的绝情人。仅仅五分钟的短片，看完后，心里却空落落的，原来寂寞是这样的平凡。一个动作，一句话，一个眼神都可能是孤独的源泉。

但是要相信，这个世界是美丽的，这只猫是乐观的，他说“我想我们都会喜欢上这个世界的”。柚编联想到的是几米的绘本——《遗失了一只猫》，因为它的结尾是：她听见了久违的一声“喵~”。
