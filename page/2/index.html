蒸汽男孩/スチームボーイ/Steamboy

发表于 2014-12-28   |   分类于 小草熊编 ， 2000~2010   |     |   阅读次数


小草熊编的说：

《蒸汽男孩》想必对蒸汽朋克狂热爱好者来说，这是一场绝无仅有的狂欢吧，画面烧到没话说，细节所到之处无不一一完美表现，用极致来形容《蒸汽男孩》的画面真心一点也不为过。重置的《FATE留过夜》在这一刻绝对是小巫见大巫了，无愧于日本史上最烧经费的动画之一。

只可惜小编我不是一个蒸汽朋克狂热爱好者，在经费燃烧肆虐过后，恢复理智，这部片子是一部不科学的科学片， 蒸汽城之中的设定大概有且只有大友克洋自己才懂，由于小编压根就不明白那个世界里用的是什么黑科技，后半段小编我就干瞪着眼睛看着掉渣天的爷爷和更掉渣天的爸爸浑然不顾国家安全秀着自己的科技研发成果，还有孙子在一旁凭借着强大的微操飞来飞去却总也死不掉，最后大家一起转转闸门，画面经费在这之后突然达到最高峰，意料之中的主人公雷喜得大小姐一只，就草草收尾了。这突如其来的一切小编我的智商立刻变得像被狗吃了一样。

但话说回来再一次看其制作画面，是个闹剧又怎么样，我想实在很难再有动画能匹敌蒸汽男孩的画面制作了，当然要超越24亿的经费的确也是个很大的问题。

看完本片最大的遗憾是没能在剧中见着三个蒸汽球一起工作时完整的蒸汽城样子，如果那真的存在存在，那它一定是一个足以匹敌地外科技的bug般的存在了。

钓球/つり球/Tsuritama

发表于 2014-12-22   |   分类于 小草熊编 ， 2010~2020   |     |   阅读次数


小草熊编的说：

大概就是在前几天，数码宝贝放出了新版人物造型海报，点心编欣喜的戳给我看，这一股满满的钓球风扑面而来，于是小编我拍案决定，这次就出钓球了。

钓球，无论从名字还是画风，让观众们眼前一亮是绝对没有问题的，但要深究其中的设定与故事，小编觉得并不算神作。但是！但是对于该番的颜色，小编实在是喜欢的不行，毕竟这是一个看脸的社会，色彩画面什么的并不会影响小编的追番进度，但绝对能加不少分，尤其第一集对男主在课堂上自我介绍时紧张而描绘的窒息感，2年后的今天仍然令人记忆犹新。

12年的小编还时刻与高中的同胞们斗智斗勇着，每周末抽出20分钟看钓球实在惬意得不行，又一周的艰苦日子熬过去了，世界也依然还是一副和平的景象，未来也许会像钓球那样美丽吧，心里这样想着，嗯，可以安心回学校了。

虽然现在钓球的剧情已经差不多忘了，只记得组长的声线异常带感，白毛有点半男不男的感觉，里面有一只鸭子，男主超有代入感，还有，那里的世界真的很美丽。

猫汤/ねこぢる草/Cat Soup

发表于 2014-12-01   |   分类于 小草熊编 ， 2000~2010   |     |   阅读次数

小编的说：

（又是好久没更新了，最近几天也是累的可以，感谢大家喜欢我们kulucolo的推荐:>

在这个寻求差异性的年代，猫汤，可以说是一部相当有代表性的OVA，汤浅魔性的画风注定让他的作品只能被他自己所超越。在我看来，猫汤就是一部童话故事，和大多数小时候听的那些童话相似的，猫汤讲述的是一个猫弟弟历尽千辛万苦为姐姐找回另一半灵魂的故事。嗯，基本这样概括就行了。但却和以前童话不相符的是猫汤中充斥着整个画面的黑暗与暴力。色调昏暗，人设随性，没有血渍乱飞却有心中时时被抽干的痛，没有言语但仿佛音乐中都带着哭泣。要问为什么猫汤总是渗透着一种不和谐，大概是它总是与我们的常识相违背着，但却又那么的相似，生与死的界限仿佛是那么的脆弱，伤口与疼痛随时随刻都会突然出现在我们身边，周遭环境毫无征兆的改变没有一点妥协的余地。没错，应该是一部面向全年龄段的童话，不用玩味你能从中看懂什么，如果不想深究，单纯的将其当作一件艺术品欣赏罢了

有时，我们大概也会为心中那一到两个天马行空的创意而进入贤者模式，但将那份心中抽象的思维具体的表现在一部动漫中，着实是一件富有魅力的事，回头想想小编我总是不能很好的将心中所想用言语表达时的那个囧样，能够这样窥视到佐藤龙雄和汤浅心中的世界，也是一种说不出幸福吧

（Ps：98年《猫汤》的作者桥口千代美的自杀令猫汤更加披上了一层迷雾，27话TV版的猫汤才更接近原著漫画

发表于 2014-10-09   |   分类于 点心编 ， 2010~2020   |     |   阅读次数

点心编的说：

在看最后一集的ED时，我的心是像上了马达一样加速跳着的。明明是什么都没有交代，用他们的日常作为结局，我却还是激动地无可复加。13集。完全的不够！

比企谷八幡。人送外号大老师。本作男主角。宅男，死鱼眼，存在感爆零点。以自己的方式活着，不去改变，蔑视着那些聚在一起的人们，把自己贬为社会的最下层。就是这样一个角色，我，真的，喜欢他。 他的那些让人觉得无言以对的谬论真的是句句直入人心。【“努力”是这个世界上最虚伪的东西之一，是成功者的借口，是失败者的安慰。然而不幸的是，努力却又是必须的，是即使明知会血本无归也不得不付出的投资。】、【大家互相帮助，一起成功，一起幸福什么的，不过是种理想。现实中，有人幸福，就必定有人被抛弃，有人光鲜，就必须有人满身泥泞。】他知道所有的人心，他敢于直面那些阴暗，不去改变自我戴上一个虚伪的面具迎合众人，在把自己定义为最下流的同时又是爱着自己的。

真的，他很勇敢，走上了一条大家都不会选择的一条路。 【对我善良的人对其他人一样善良，而我却总会忘掉这一点，如果真相是残酷的，那么谎言肯定就是善良的，所以善良就是谎言。总是期待着，总是误会着。不知从何时开始我就放弃了希望。训练有素的孤零零不会再上同样的当。经历百战的强者，说到打败战我是最强的】大老师字字入心坎。

所以我是讨厌叶山的。那种对所有人都温柔，看似大家都受到了温暖，却是对所有人都不公平的。让人有了妄想，殊不知自己和他人无二样。让别人在这种温暖中沉溺，虽说是无意，但我是很厌恶的。所以我觉得大老师才是真正温柔的，那种以伤害自己来帮助他人的温柔。那种别人都做不到的温柔。 侍奉部的故事还在继续着。

我的青春恋爱喜剧果然有问题。 到底是哪里出了问题呢【笑
