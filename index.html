秒速五厘米/秒速5センチメートル

发表于 2016-06-09   |   分类于 小草熊编 ， 2000～2010   |     |   阅读次数

小编的说:

懂了很多道理，但还是过不好这一生。

只是平凡的过着日子， 悲伤就会聚集在周围。

这不是我第一次看秒五了，但，每一遍都是新的心的体验。眼看着明理和贵树随着时间线的一点点变化，从爱的初心到无力的反抗再到看似再正常不过的妥协。现在有太多的作品让我们对成人后的一切感到抗拒了，而他们总能精准的放大内心的悲伤。

若愿望能够实现，我要立刻到你的身边。一句 One More Time 的歌词，却在多数情况下只能是一句歌词，在当你做出某个决定的那一瞬间，你还是你，而愿望注定只能永远成为愿望。没错，我们都是这样认为的，但也总是会这样而相互错过。错过不分对错，就像改变也不分好坏，正是一次次源自内心挣扎过后的选择堆积成了的我们每个独立的人格。秒五所怀念的也许只是那段无法被拯救的初恋，但人生可不止这么简单，它往往更残酷，更煎熬，当然也注定远远更美丽。快乐与悲伤往往也就一念之差，我们悲伤是因为容易怀念起那段美丽的日子，然而当拾起美丽过后那一段痛苦，你会发现，你也已经可以了。

大都会/メトロポリス/Metropolis

发表于 2016-02-26   |   分类于 小草熊编 ， 2000~2010   |     |   阅读次数

小草熊想说：

很早就有听说过大友克洋的大都会，记忆中是他的三部曲之一，被奉为经典。这段时间的确是很久没有正正经经的看完一部动漫了，如今看动漫更多像是跟着自己的直觉，猛然间回头发现作品太多，没必要像是刻意的达成一般去解锁自己的二次元成就，唯一有说服力的就是个足够有实力吸引我的开头，没想到这次回归的契机却是泡下一桶开杯乐的功夫。一直以来口味就挺独特，所以并不对这突如其来的画风感到有所不适，结果当蒂玛出场的那一刻我竟然被ta折服了。自那之后怎么看怎么觉得这种阿童木式手冢画风真是一级棒。表情流畅举止自然精致至极外加巨精致的背景，看得我完全无法用言语来形容出这种享受。虽然这已经是一部2000后的作品, 但依然完全足以体现出那个时代的动画的魅力之大。出色的分镜独具性格的配乐，看得我一乐一乐的。我们都无法给出对于一个作品绝对客观的评价，因为撇去自己的喜好，我们或多或少也都结合进了当时观看时期的内心感受。大都会之后，我是感到了些许的悲伤，虽然最终一幕那个世界很静很明亮很美好，但是我却已经开始对无法去那个世界感到失落了。爱与恨的都如此刻骨，执着的人至始至终执着，善良的人至始至终善良，而女主，真的很美丽。
